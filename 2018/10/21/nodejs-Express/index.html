【NodeJS】Express框架

2018-10-21
FrontEnd

Hello world

构建最简单的express app

var express = require('express');
app = express();

app.listen(3000,()=>{console.log('app start!');});

路由

express路由形式：app.method(PATH, handler), method包括post、get、put和delete。
app.all(PATH, handler), 可以反应四种请求
路由路径写为/:title，则在请求/anything的时候，在req.param.title中可以获取到anything的值

静态文件

express内置中间件函数express.static()，app.use(express.static('public'))将public目录对外开放访问。
在程序中可以直接用css/style.css的方式直接引入样式。

获取get和post的参数

Get参数直接使用req.query.访问。
Post参数直接使用req.body.访问。

cookie & session

设置cookie首先需要引入cookie-parser模块，然后可以直接通过 res.cookie(name, value, [options]) 进行设置，若需要使用{signed: true} 选项，则需要在引入中间件cookie-parser时传入secret key: app.use(cookieParser(secret))。
设置session需要引入express-session模块，首先设置：

app.use(session({
  secret:'y4ngyy',
  saveUninitialized: false,
  resave: false,
  cookie: {
    maxAge: 60000,
    httpOnly: true
  }
}));

然后登陆时可以如下使用:

req.session.regenerate(function(err) {
    if(err) {
        return res.json({status: false, err:'Something wrong in session'});
    }
    req.session.username = result[0].username;
    res.json({status: true});
});
